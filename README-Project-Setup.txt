Unzip file into a directory.
Open the TW2-com.example.FizzBuzzProcessor folder as an IntelliJ project
File | Project Structure | Project | SDK  (Choose "coretto-21" in dropdown list, or EDIT button, "+" to add New SDK via download) "Amazon coretto 21.0.2"
Set the SDK to coretto-21 - that works, and is the one I tested it with. 
Set Language Level to 20. 

Project folders and Maven Dependencies:
   pom.xml sets [Sources Root] to "src" and [Test Sources Root] to "test" folders.
   pom.xml specifies version of JUnit to be 5.10.2
   No changes should be required.
